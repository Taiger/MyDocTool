How to Edit Sliders

There are two ways to edit the sliders: on the actual slider with a contextual link or off of the main slider list.

  1. This contextual link can be hard to find unless you have a big monitor. The window width has to be larger than the slider width (1920px).

2) The slider list is where you can add new sliders. It is at at this path: /admin/structure/fieldable-panels-panes/manage/slider-revolution

which can be found off of the admin menu under Structure -> Fieldable Panel Panes -> Slider revolution.

This is the Slider List. You can add/edit/delete or preview sliders from here.
