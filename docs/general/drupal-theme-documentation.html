Drupal Theme Documentation

Living documentation for the platform theme, AF1, and site-specific themes such as ASP3

 

To process all less files in both the platform and site-specific themes, add this as an alias:

alias makecss = "pushd sites/all/themes/af1; grunt;cd ../asp3;grunt;echo 'done.';popd;"

Platform Theme

 

Structure of Less/Css

Annenberg Space for Photography v3 - ASP3

Site Theme for the Annenberg Space for Photography

 

Structure of Less/Css

Style.less contains all of the less files, including the Bootstrap library. The platform less files from AF1 are also included. This is preferable as they can be processed together.
