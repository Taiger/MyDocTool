What is this site?

This the styleguide and documentation website for ASP3.0.

It is powered by Codeignitor. I uses a number of components borrowed from the Styleguide Biolerplate.

Features:

  1. login/logout
  2. create/edit/delete Docs across two categories.
  3. create/edit/delete Patterns across five categories.
  4. WYSIWYG customized for Bootstrap powered by CKeditor.
  5. HTML for Patterns editable with the Ace editor.
  6. Basic page caching for extra speedy pages
  7. Styling is namespaced to seperate from it from the Less files.
  8. Less is processed to Css using Grunt
