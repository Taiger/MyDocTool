Component Naming

A hybrid approach

Components are composed of Atoms (html elements) and sometimes Molecules (fields). Multiple Components are put together to form Templates. Templates are assigned to Pages.

Component Class Names

Components should be written in Capitol Casing or UpperCamelCase. This is to differentiate them from Drupal classes and Bootstrap classes, which both use dashes and lowercase.

ImageHero
VideoHero
CarouselHero

Component Parts

Each component part, is named using double dashes extended from the base component. Component Parts are totally optional, as they don't always make sense. Note that is this is mot Molecules or Atoms, as they are always reusable.

ImageHero--head
ImageHero--content
ImageHero--footer

Component Modifiers

A variation of Component is named by extending the base Component name with double underscores.

ImageHero__special
ImageHero__secondary
ImageHero__secondary--head
